## Vite+ workflow notes

- Install dependencies with `vp install`.
- Run lint, formatting, and type checks with `vp check`.
- Run tests with `vp test`.
- Run repository scripts with `vp run <script>`, for example `vp run build` or `vp run dev`.
- Build the CLI SEA binary with `vp run build:binary`.
